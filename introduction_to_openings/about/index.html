About

This page is solution to second assignment for WWW apps course. It presents use of tailwind to generate css.

The project focuses on applying Tailwind's utility-first approach to style a webpage, demonstrating various features such as text and background colors, box shadows, animations, and table styling. Additionally, it explores responsive design techniques using flexbox utilities

Through this assignment, we aim to showcase how Tailwind CSS simplifies frontend development while maintaining flexibility and performance.
